Header 

 

The header class is used to make the elements at the top of the page line up. It keeps the elements at the top of the page when you scroll down. This way you can always see the header.

The logo is lined up with the title. They are centred. The logo class is used to make the logo look a way. It makes the logo a certain size gives it a shape and puts a border around it.
